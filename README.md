# 🚚 Transport Module - MERN Stack App

A Transport Management System built with the MERN Stack for a Manufacturing Company.

## Features
- 📊 Dashboard with Charts & Statistics
- 🚗 Vehicle Management (Add, Update, Delete)
- 📦 Delivery Scheduling & Tracking
- 📍 Live Delivery Status Timeline
- 🔍 Search & Filter
- 📄 Export PDF & CSV Reports

## Tech Stack
- **MongoDB** - Database
- **Express.js** - Backend Framework
- **React.js** - Frontend
- **Node.js** - Server

## Setup Instructions
1. Clone the repo
2. Add your MongoDB URI in `backend/.env`
3. Run `npm install` in both `backend` and `frontend`
4. Run `nodemon server.js` in backend
5. Run `npm start` in frontend
